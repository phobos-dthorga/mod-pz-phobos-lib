# Project Identity Policy (PhobosLib)

The name "PhobosLib" represents the official maintained release of this shared utility library.

Allowed:
- Fork and modify
- Redistribute modified versions
- Bundle as a dependency of other mods (recommended)

Required:
- Clearly label forks as unofficial
- Provide attribution to the original author

Not allowed:
- Reupload unchanged under the same name
- Imply official status or endorsement of unofficial forks
- Cause confusion about authorship or the “canonical” release

Official distribution: the GitHub repository and/or Steam Workshop entry maintained by the original author (if applicable).
