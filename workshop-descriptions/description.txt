[h1]Phobos Shared Library (PhobosLib)[/h1]
A shared utility library for Project Zomboid Build 42 mods. PhobosLib provides reusable systems that other Phobos mods depend on — [b]it adds no gameplay content on its own[/b].

If you're subscribing to [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3668197831][b]Phobos' Chemistry Pathways[/b][/url], you need this mod installed.

[hr][/hr]
[h2]What It Provides[/h2]
8 utility modules covering common PZ modding needs:

[img]https://raw.githubusercontent.com/phobos-dthorga/mod-pz-phobos-lib/main/docs/images/phoboslib-modules.png[/img]
[i][url=https://github.com/phobos-dthorga/mod-pz-phobos-lib/tree/main/docs]Full API reference on GitHub[/url][/i]

[list]
[*][b]Util[/b] — Safe method calling, API probing across Build 42 variants, keyword item search, modData helpers
[*][b]Fluid[/b] — Build 42 fluid container helpers with multi-strategy API fallbacks
[*][b]World[/b] — Tile scanning, object search, generator and vehicle detection
[*][b]Sandbox[/b] — Safe sandbox variable access, runtime mod detection, yield multipliers
[*][b]Quality[/b] — Quality/purity tracking: 0-100 scoring, tier lookup, equipment factors, yield penalties
[*][b]Hazard[/b] — PPE detection, respiratory protection, mask filter degradation, EHR disease dispatch
[*][b]Skill[/b] — Perk queries, safe XP awards, cross-skill XP mirroring
[*][b]Reset[/b] — Deep inventory traversal, modData stripping, recipe removal, XP reset, item cleanup
[/list]

[h2]For Mod Developers[/h2]
PhobosLib is designed as a stable API dependency for your own PZ mods. All public functions follow semantic versioning with backwards compatibility guarantees.

[list]
[*]Full API reference with function signatures: [url=https://github.com/phobos-dthorga/mod-pz-phobos-lib/tree/main/docs]Documentation[/url]
[*]Contribution guidelines, versioning policy, and architecture: [url=https://github.com/phobos-dthorga/mod-pz-phobos-lib]GitHub Repository[/url]
[*]File bugs or request new utilities: [url=https://github.com/phobos-dthorga/mod-pz-phobos-lib/issues]GitHub Issues[/url]
[/list]

Usage:
[code]require "PhobosLib"
local qty = PhobosLib.Quality.getQuality(item, "purity", 50)
local hasMask = PhobosLib.Hazard.getRespiratoryProtection(player)
[/code]
